# Projeto Integrador - Haru Edu BI

Este projeto consiste em uma plataforma de Business Intelligence (BI) para a empresa **Haru Edu**, polos de apoio da **Unoeste**. O objetivo é transformar dados fragmentados em visualizações estratégicas para a tomada de decisão.

## 🛠️ Tecnologias Utilizadas
* **Banco de Dados:** SQL Server (Armazenamento e estruturação).
* **BI:** Power BI (Criação de Dashboards e medidas DAX).
* **Backend:** Python com Flask (Sistema de login e servidor web).
* **Frontend:** HTML5 e CSS (Bootstrap) para a interface do usuário.

## 📂 Estrutura do Projeto
* `/sql_scripts`: Scripts de criação de tabelas e carga de dados.
* `/templates`: Arquivos HTML do portal.
* `app.py`: Lógica do servidor e controle de acesso.

## 🚀 Como executar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o servidor: `python app.py`
3. Acesse: `http://127.0.0.1:5000`
